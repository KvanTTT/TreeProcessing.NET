TreesProcessing.NET
==========================================
Thee structures serialzation, traversal and comparison on .NET platform.

## Description

Already implemented features:
* Serialization:
    * Json.NET FullTypeName.
    * Json.NET Property.
    * Json.NET ClassName.
    * Json.NET Attribute.
    * Protobuf.
* Visitor (Static, Dynamic).
* Listener (Static, Dynamic).
* Comparison (Static, Dynamic).
